Data collected when visiting the website

When you access this website, your browser automatically sends certain technical information to GitHub’s servers. This may include:

  • Your IP address
  • The date and time of access
  • The URL or file requested
  • The website you came from
  • Your browser type and version
  • General information about your operating system

These server logs are created and processed by GitHub. They are used to ensure the stability, security, and proper functioning of the hosting service. I do not have access to individual log files and do not process this data independently.

Joint controllership with GitHub

According to the case law of the European Court of Justice, operators of websites that integrate third-party infrastructure may be considered joint controllers with those providers regarding certain processing activities. For the hosting and delivery of this website, GitHub and I act as joint controllers for the initial collection of technical data (such as IP addresses) transmitted by your browser.

GitHub processes this data under its own responsibility for security and operational purposes.
For more details, see Github's privacy statement here: https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement

Legal basis

The processing of technical data such as IP addresses when loading this website is based on legitimate interest (Art. 6(1)(f) GDPR). This includes ensuring the secure and reliable operation of the website.

International data transfers

As part of operating GitHub Pages, technical data may be processed on servers outside the European Union. GitHub uses safeguards such as Standard Contractual Clauses and participation in the Data Privacy Framework to ensure an adequate level of data protection.

How long GitHub stores log data

Log retention periods are determined by GitHub and may vary depending on the type of log (e.g., debugging, security, operational needs). Since GitHub Pages does not provide retention controls to site owners, the duration is based solely on GitHub's policies.

For detailed and up-to-date information, please consult GitHub's privacy documentation or GitHub support.
