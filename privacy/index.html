Privacy Policy

This website is a purely open source, static site. It uses no cookies, no analytics, no trackers, and stores no personal data. No third-party scripts are used and no data is transmitted to external services. All resources such as source files and media are served locally. The only external service involved is GitHub Pages, which hosts and delivers the static content.

Hosting via GitHub Pages

When you access the website, your browser automatically transmits certain information to GitHub's servers so that the requested page can be delivered. This includes:

  • IP address of the requesting device
  • Date and time of access
  • URL or file requested
  • Referrer URL
  • Browser type and version
  • Operating system information

These data are processed and stored solely by GitHub in server log files. The purpose is to maintain the technical operation, security, and stability of the hosting service and to detect or prevent disruptions or misuse. For further information, please refer to GitHub's Privacy Policy.

Contact

If you contact us via email, any personal data you voluntarily provide will be used solely to process your inquiry. Your data will not be shared with third parties.
